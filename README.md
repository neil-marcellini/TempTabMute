# TempTabMute

A Chrome extension that mutes the current tab for a chosen duration, then automatically unmutes it.

## Features

- One-click mute with a configurable timer
- Live countdown showing remaining mute time
- Settings page to pick your preferred duration (30s, 1 min, 5 min, 15 min, 30 min, 1 hour)
- Auto-unmute when the timer expires
- Manual "Unmute Now" button

## Install as Unpacked Extension

1. Open Chrome and go to `chrome://extensions/`
2. Enable **Developer mode** (toggle in top-right)
3. Click **Load unpacked** and select this project folder
4. The TempTabMute icon appears in your toolbar — click it to use

## Usage

1. Navigate to any tab playing audio
2. Click the TempTabMute extension icon
3. Press the **Mute** button to mute for the configured duration
4. Watch the countdown — the tab unmutes automatically when it hits zero
5. Click **Settings** to change the default mute duration
